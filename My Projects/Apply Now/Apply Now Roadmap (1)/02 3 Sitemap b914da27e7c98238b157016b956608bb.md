## 0. Authentication

- **0.1 Login / Single Sign-On (SSO)**
- **0.2 Forgot Password**

## 1. Global Navigation (Main App Menu)

The persistent left-hand or top navigation bar accessible from anywhere in the app.

- **1.1 Global Dashboard (Home)**
    - 1.1.1 Overview of Active Campaigns (Widgets)
    - 1.1.2 Pending Tasks / @Mentions
    - 1.1.3 Quick Action: [Start New Campaign]
- **1.2 Jobs / Campaigns**
    - 1.2.1 Create New Campaign (Wizard: Job Details, Dealbreaker setup)
    - 1.2.2 Active Campaigns Directory
    - 1.2.3 Archived / Closed Campaigns
- **1.3 Global Talent Vault** *(HR Director Focus)*
    - 1.3.1 Cross-Campaign Candidate Search
    - 1.3.2 Saved Talent Pools
    - 1.3.3 GDPR & Compliance Logs
- **1.4 Reports**
    - 1.4.1 Pipeline Speed (Time-to-Hire)
    - 1.4.2 Sourcing Analytics
- **1.5 Settings**
    - 1.5.1 Team & Permissions Management
    - 1.5.2 Templates (Emails, Scorecards, Standardized Rejection Reasons)

## 2. Campaign Level (Drill-down from Jobs > Active Campaigns)

When a user clicks on a specific job (e.g., "Senior React Developer").

- **2.1 Campaign Overview** (Metrics for this specific role)
- **2.2 CANDIDATES LIST**
    - **2.2.1 Filter & Search Pane** (Location, Right-to-Work, Keywords)
    - **2.2.2 Pipeline Stage Views** (New Applied, Phone Screen, Interview, Offer)
    - **2.2.3 Bulk Actions Menu** (Bulk Reject, Bulk Advance, Send Assessment)
- **2.3 Job Setup / Configuration** (Edit job description, adjust hiring team)

## 3. Candidate Level (Drill-down from Candidates List)

When a user clicks on a specific candidate row from the list.

- **3.1 CANDIDATES INFO PAGE** 📍 *(Primary Design Deliverable #2)*
    - **3.1.1 Left Panel (Candidate Context)**
        - Inline Resume / PDF Viewer (Native rendering)
        - Quick Contact Details (Phone, Email)
        - External Links (LinkedIn, Portfolio, GitHub)
    - **3.1.2 Right Panel (Evaluation Workspace)**
        - **Tab A: Scorecard** (Standardized rating criteria)
        - **Tab B: Team Notes** (Rich text editor with @tagging functionality)
        - **Tab C: History** (Past applications, previous pipeline stages)
        - **Tab D: Emails** (Direct communication thread with candidate)
    - **3.1.3 Header Quick Actions**
        - Change Stage (Advance/Reject)
        - Quick Rating (Thumbs Up/Down)
        - Share Profile
