# **CareerPulse: AI-Powered Job Decision Engine**

## **🌍 The Problem**

The modern job search is broken. Candidates are overwhelmed by an ocean of job postings that are often filled with opaque corporate jargon, vague responsibilities, and unlisted salary ranges.

When evaluating an opportunity, candidates constantly struggle with critical questions:

- *Is this job actually a good fit for my skills?*
- *What does this company truly do, and what is its culture?*
- *What should I realistically expect from the day-to-day of this role?*
- *Is leaving my current path worth the risk for this specific opportunity?*

Without clear answers, job seekers suffer from decision fatigue, often applying blindly or passing up on roles that could have been perfect fits. The process lacks objective, data-driven analysis tailored to the individual.

## **🧗 The Challenge**

How can we parse unstructured, heavily stylized job descriptions from any source and instantly cross-reference them against a candidate's unique professional history and deeply personal preferences (like target salary, working models, and desired tech stacks)?

Furthermore, how can we present this complex analytical breakdown in a UI that is empowering and intuitive, rather than overwhelming to an already stressed job seeker?

## **💡 The Hypothesis / Solution**

**Hypothesis:** If we provide candidates with a localized, AI-driven evaluation tool that cross-references their resume and preferences against raw job descriptions, we can drastically reduce application anxiety and improve interview success rates by offering objective "Match Scores" and actionable insights.

**The Solution:** Enter **CareerPulse**. A personalized, intelligent web application that acts as a private career advisor. Candidates input their profile and preferences once. Then, for every job description they encounter, CareerPulse utilizes Large Language Models (LLMs) to cut through the fluff and deliver a structured risk/reward analysis.

---

## **✨ Core Features & Functionality**

### **1. Instant Job Decoding (The "Career Advisor")**

Users simply paste any raw job description directly into the app. CareerPulse parses the text using AI to identify the core responsibilities, hidden expectations, and specific technical requirements, translating "HR-speak" into plain language.

### **2. Objective Match Scoring**

The application doesn't just summarize the job; it evaluates it. By checking the parsed job requirements against the user's saved CV and settings (Target Salary, Tech Stack, Remote/Hybrid preference), it generates a definitive **Match Score (0-100%)**. It clearly highlights where the user exceeds requirements and flags potential friction points.

### **3. Deep Company Insights**

To answer the "What is this company about?" question, the AI generates a brief on the hiring company, providing the candidate with critical context about the industry, the company's presumed culture based on the posting's tone, and realistic expectations.

### **4. Application Tracking & History Dashboard**

A built-in CRM for the job hunt. All analyzed jobs are automatically saved to a history dashboard. Candidates can:

- Filter by application status (e.g., Applied, Interviewing, Rejected)
- Sort by their personal Match Score
- Save private interview notes and prep materials alongside the original job analysis.

## **🛠️ Technical Architecture**

CareerPulse is built with a modern, fast, and scalable frontend-first stack.

### **Frontend**

- **Framework:** React 18 + Vite for lightning-fast HMR and optimized production builds.
- **Language:** TypeScript for strict type safety and self-documenting code.
- **Styling & UI:** TailwindCSS for rapid, utility-first styling. The UI focuses on a premium aesthetic with subtle animations, glassmorphism touches, and a highly responsive layout.
- **Animations:** `framer-motion` powers the fluid interactions, including a completely custom-built, continuously gliding Onboarding Wizard portal that uses real-time DOM tracking instead of traditional unmounting modals.
- **Icons:** `lucide-react` for crisp, consistent vector iconography.

### **Backend & Cloud Services**

- **AI Engine:** Google Gemini API (`@google/genai`). The application utilizes Gemini to process the unstructured job text and return heavily structured JSON responses that populate the UI dashboards. Prompt engineering ensures the AI focuses strictly on actionable career advice.
- **Authentication:** Firebase Auth handles secure user sign-ups and logins, supporting seamless user session management.
- **Database (Planned/Current):** Firebase Firestore/Local Storage manages the persistence of the user's profile, preferences, and the history array of analyzed job postings.

### **Deployment & CI/CD**

- **Hosting:** Vercel automatically deploys the `main` branch from the GitHub repository, utilizing Vite's optimized build output and securely injecting runtime environment variables (like the Gemini and Firebase keys).
